## 📝 Sobre o Projeto
Este é um projeto fundamental desenvolvido para consolidar os conceitos teóricos de como uma **API** funciona.

Embora o sistema não realize chamadas externas para um servidor na nuvem, ele simula o comportamento de uma API real através do consumo de uma estrutura de dados local (JSON). O foco foi aprender como os dados trafegam entre o "back-end" simulado e a interface do usuário.

### 🎓 Objetivos de Aprendizado
* **Estrutura JSON:** Entender como os dados de participantes (nome, foto, status) são organizados.
* **Manipulação de Array de Objetos:** Como iterar sobre dados para criar elementos HTML dinamicamente.
* **Lógica de Front-end:** Separar a camada de dados da camada de apresentação.


## 🛠️ Funcionalidades Simuladas
* **Renderização Dinâmica:** O JavaScript percorre a "base de dados" e cria os cards dos participantes automaticamente.
* **Filtro de Dados:** Lógica para exibição de informações específicas de cada integrante do BBB21.


## 💻 Tecnologias Utilizadas
* **HTML5:** Estrutura da página.
* **CSS3:** Layout e estilização dos cards.
* **JavaScript:** Lógica de consumo e renderização dos dados locais.
